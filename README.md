# ai-dev-sandbox

Repository snapshot: documentation-first sandbox for AI/ML experimentation using Docker on Windows.

Overview

- Target environment: Windows 11 with Docker Desktop (WSL2 backend).
- Scope: LLM inference, Python ML training (PyTorch/TensorFlow), API/backend services, Jupyter notebooks, and IIS-hosted apps.
- GPU: Supports NVIDIA and AMD GPU passthrough paths (choose per hardware).
- Key steps: prerequisites & system check, enable WSL2, install Docker Desktop, GPU configuration, performance tuning, security hardening, workload container setups, and monitoring.

For full, step-by-step setup instructions and configuration examples, see: docs/docker-ai-sandbox-guide.md

How to use

- Read the docs/ guide for detailed installation and tuning steps.
- Add project source files at the repo root or a language-specific subfolder.

Contributing
